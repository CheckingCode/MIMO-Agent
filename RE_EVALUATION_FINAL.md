## 四、优缺点总结

### MiMo Agent 优势

| 优势 | 说明 |
|------|------|
| **智能输入预处理** | 自动修复错别字、澄清意图，Claude 没有 |
| **专家人格系统** | 7 个角色自动匹配，比 Claude 的通用 prompt 更精准 |
| **对抗模式** | Brain A+B 双脑审核，质量更高，Claude 没有 |
| **Docker 沙箱** | 命令在隔离容器执行，比 Claude 更安全 |
| **多模式支持** | Auto/Polling/Plan/Red Team，Claude 只有单一模式 |
| **中英文切换** | 一键切换，Claude 无多语言支持 |
| **模型智能降级** | 图片自动切换 vision 模型，Claude 无此功能 |
| **上下文主动管理** | Token 估算+滑动窗口，防止爆上下文 |
| **极致轻量** | 113 KB VSIX，Claude 数 MB |
| **任务进度可视化** | Update Todos 组件，实时显示进度 |
| **MCP 支持** | 标准化工具扩展，与 Claude 对等 |
| **专家人格自动切换** | 根据任务类型自动选择最佳角色 |

### MiMo Agent 劣势

| 劣势 | 说明 |
|------|------|
| **无浏览器工具** | 不能打开浏览器、点击、截图 |
| **无测试覆盖** | 没有单元/集成测试 |
| **无历史搜索** | 不能搜索历史对话 |
| **无导出功能** | 不能导出对话为 Markdown/JSON |
| **Diff 视图基础** | 没有并排对比视图 |
| **模型能力依赖** | 底层模型能力决定上限 |
| **开源生态弱** | 闭源，无社区贡献 |

---

## 五、结论

> **MiMo Agent v1.3.0+ 在功能完成度上已达 92%，在多个维度上超越了 Claude Code。**
>
> **核心优势**：智能输入预处理、专家人格系统、对抗模式、Docker 沙箱、多模式支持 — 这些是 Claude Code 完全没有的功能。
>
> **核心差距**：仅剩浏览器工具、测试覆盖、历史搜索、导出功能等非核心功能差距。
